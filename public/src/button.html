Button Component의 Prototype

basic button

button은 다음과 같이 생성합니다

button은 button markup에 button class를 사용합니다

Markup & Example
<button class="ui button">Basic Button</button>

Prototype

<Button></Button>

emphasis

강조가 필요한 Button에 emphasis속성을 사용합니다

사용가능한 속성은 아래와 같습니다

primary
secondary
default(생략가능)
Markup & Example
<button class="ui primary button">Primary</button>
<button class="ui secondary button">Secondary</button>
<button class="ui button">Basic</button>

Prototype

<Button emphasis="primary"></Button>
<Button emphasis="secondary"></Button>
<Button></Button>
