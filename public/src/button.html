Button

Button Component의 Prototype

basic button

button은 다음과 같이 생성합니다

button은 button markup에 button class를 사용합니다

Markup & Example
<button class="ui button">Basic Button</button>

Prototype

<Button></Button>

emphasis

강조가 필요한 Button에 emphasis속성을 사용합니다

사용가능한 속성은 아래와 같습니다

primary
secondary
default(생략가능)
Markup & Example
<button class="ui primary button">Primary</button>
<button class="ui secondary button">Secondary</button>
<button class="ui button">Basic</button>

Prototype

<Button emphasis="primary"></Button>
<Button emphasis="secondary"></Button>
<Button></Button>

labeld/icon

Label/Icon Button에 labeld/icon속성을 사용합니다

사용가능한 속성은 아래와 같습니다

labeled
icon
labeledIcon
1024
Markup & Example
<div class="ui labeled button">
	<div class="ui button">
		<i class="heart icon"></i>
	</div>
	<a class="ui basic label">1024</a>
</div>
<button class="ui icon button">
	<i class="cloud icon"></i>
</button>
<button class="ui labeled icon button">
	<i class="right arrow icon"></i>
	Next
</button>

Prototype

<div class="ui Button" labeld="1024" icon="heart"></div>
<Button class="ui icon Button" icon="cloud"></Button>
<Button class="ui labeled icon Button" labeledIcon="right arrow">Next</Button>

States

Button의 다음과 같은 States 속성을 사용합니다

사용가능한 속성은 아래와 같습니다

active
loading
Markup & Example
<button class="ui active button">
	<i class="user icon"></i>
	Follow
</button>
<button class="ui disabled button">
	<i class="user icon"></i>
	Follow
</button>
<button class="ui loading button">
</button>

Prototype

<Button class="ui Button" icon="user" active="true">Follow</Button>
<Button class="ui Button" icon="user" active="false">Follow</Button>
<Button class="ui Button" loading="true"></Button>
