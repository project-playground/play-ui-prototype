Group

Group Component의 Prototype

button group

button group은 다음과 같이 생성합니다

button group은 div markup에 buttons class를 사용합니다

Markup & Example
<div class="ui buttons">
  <button class="ui button">One</button>
  <button class="ui button">Two</button>
  <button class="ui button">Three</button>
</div>

Prototype

<Buttons>
  <Button >One</Button>
  <Button >Two</Button>
  <Button >Three</Button>
</Buttons>

group direction

group 의 방향을 설정하는 속성입니다.

groupDir
Markup & Example
<div class="ui buttons">
  <button class="ui button">One</button>
  <button class="ui button">Two</button>
  <button class="ui button">Three</button>
</div>
<div class="ui vertical buttons">
  <button class="ui button">One</button>
  <button class="ui button">Two</button>
  <button class="ui button">Three</button>
</div>

Prototype

<Buttons>
  <Button >One</Button>
  <Button >Two</Button>
  <Button >Three</Button>
</Buttons>
<Buttons groupDir="vertical">
  <Button >One</Button>
  <Button >Two</Button>
  <Button >Three</Button>
</Buttons>

icon button group

icon button group은 다음과 같이 생성합니다

button group은 div markup에 icon buttons class를 사용합니다

Markup & Example
<div class="ui icon buttons">
  <button class="ui button"><i class="align left icon"></i></button>
  <button class="ui button"><i class="align center icon"></i></button>
  <button class="ui button"><i class="align right icon"></i></button>
  <button class="ui button"><i class="align justify icon"></i></button>
</div>
<div class="ui icon buttons">
  <button class="ui button"><i class="bold icon"></i></button>
  <button class="ui button"><i class="underline icon"></i></button>
  <button class="ui button"><i class="text width icon"></i></button>
</div>

Prototype

<IconButtons>
  <Button icon="align left"></Button>
  <Button icon="align center"></Button>
  <button icon="align right"></Button>
  <button icon="align justify"></Button>
</IconButtons>
<IconButtons>
  <Button icon="bold"></Button>
  <Button icon="underline"></Button>
  <Button icon="text width"></Button>
</IconButtons>
